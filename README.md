# Matrix-Edge-Detection

A Python script that detects the edges from a live feed webcam via OpenCV Canny Line detection and creates a matrix overlay.

## To run
Run "pip install -r requirements.txt" for the required libraries.
For this project, I worked with Python 3.11.7
